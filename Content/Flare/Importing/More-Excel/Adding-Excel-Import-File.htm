Adding an Excel Import File

The following steps show you how to add an Excel import file, which is used to import Excel files into a project.

how to add an Excel import file to a project
    • Ribbon In the Content section select New > Import File > Add MS Excel Import File.
    • Right-Click In the Project Organizer, right-click on the Imports folder and from the context menu select Add MS Excel Import File.

    The Add File dialog opens.

  2. In the File Type field at the top, make sure MS Excel Import File is selected.
  5. In the File Name field, type a new name for the import file.
  6. (Optional) See Applying Conditions to Content.
  7. (Optional) See Associating Tags with Files.
  8. Click Add. The import file is added to the Imports folder in the Project Organizer.
