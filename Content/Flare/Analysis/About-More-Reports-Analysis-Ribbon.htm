More Analysis Scans

In the Analysis ribbon, the More Reports drop-down button provides several report options. They include:

Duplicates
  • Duplicate Items in TOC Opens the Duplicate TOC Items window pane (if you are analyzing a Flare project), which lets you view a list of all files in your project that have been added more than once to a table of contents. See Viewing Duplicate Items in a TOC.

  • Duplicate Styles Opens the Duplicate Styles window pane. See Viewing Duplicate Styles.

 Files
  • Files with Annotations Opens the Project Annotations window pane. See Viewing Files with Annotations.
  • Files with Changes Opens the Files with Changes window pane where you can view all topics in your project that contain tracked changes. See Viewing Files with Changes.
  • Files with Snippet Conditions Opens the Topics with Snippet Conditions window pane. See Viewing Files with Snippet Conditions.
  • Files without File Tags Opens the Files Without File Tags window pane. See Viewing Files without File Tags.
 Topics
  • Non-XML Topics Opens the Non-XML Topics window pane. See Viewing Non-XML Topics.
  • Topics not in TOC Opens the Topics Not in Selected TOC window pane (if you are analyzing a Flare project). See Viewing Topics not in a TOC.
  • Topics not Indexed Opens the Topics Not in Index window pane. See Viewing Topics not Indexed.
  • Topics not Linked See Viewing Topics not Linked.
  • Topics not Linked by Map ID Opens the Topics Not Linked By Map ID window pane (if you are analyzing a Flare project). See Viewing Topics not Linked by a Map ID.
  • Topics without Concepts Opens the Topics Without Concepts window pane. See Viewing Topics without Concepts.
 Unused Items

You can open the Unused Items window pane, which lets you view lists of unused elements in your project.

  • Viewing Unused Bookmarks

  • Viewing Unused Content Files
  • Viewing Unused CSH IDs

  • Viewing Unused Styles
