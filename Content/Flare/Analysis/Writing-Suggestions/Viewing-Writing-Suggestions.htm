Viewing Writing Suggestions

how to view writing suggestions
  1. Open a project.
    • Ribbon  In the Dynamic Reports section, select Suggestions > Writing Suggestions.
    • Summary Window Pane Double-click the row that mentions writing suggestions.

    The Writing Suggestions window opens.

    • Suggestion Displays the issue (e.g., "H1s with single child," "Skipped heading"). Following are the types of issues that might be found, depending on your selections in the Writing Suggestions Search Properties dialog.

    • Tag Displays the tag where the issue occurs.
    • File
    • Folder
What's Next?

After viewing writing suggestions, you can open each file and make changes accordingly. To open the file, either double-click on the writing suggestion row, or right-click on the row and select Open.

You can also customize the writing suggestions that are displayed. See Customizing Writing Suggestions Shown.
