About Suggestions

In the Analysis ribbon, the Suggestions drop-down button provides several options. They include:

  • Index Keyword Suggestions See Viewing Index Keyword Suggestions.
  • Snippet Suggestions See Viewing Snippet Suggestions.
  • Variable Suggestions See Viewing Variable Suggestions.
  • New Style Suggestions See Viewing New Style Suggestions.
  • Local Style Suggestions See Viewing Local Style Suggestions.
  • Accessibility Suggestions See Viewing Accessibility Suggestions.
  • Markup Suggestions See Viewing and Applying Markup Suggestions.
  • Cross-Reference Suggestions See Viewing Cross-Reference Suggestions.
  • Writing Suggestions See Viewing Writing Suggestions.
  • Frequent Segments See Viewing Frequent Segments.
  • Similar Segments See Viewing Similar Segments.
  • Segments for Topics You can view a list of topics associated with a segment (as selected in the Frequent Segments window pane or Similar Segments window pane). This is simply a quick way to open a file where a particular segment exists. See Viewing Segments for Topics.
