About Used Items

In the Analysis ribbon, the Used Items drop-down button provides several options. They include:

  • Used Condition Tags You can view a list of all condition tags that have been applied in files throughout your project. See Viewing Used Condition Tags.
  • Used File Tags See Viewing Used File Tags.
  • Used Language Tags See Viewing Used Language Tags.
  • Used Variables See Viewing Used Variables.
  • Used Index Keywords See Viewing Used Index Keywords.
  • Used Concepts See Viewing Used Concepts.
  • Used Bookmarks See Viewing Used Bookmark Links.
