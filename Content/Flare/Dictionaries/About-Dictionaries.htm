 Custom Global and Project Dictionaries

See Spell Checking While Typing and Spell Checking Manually.

There are two kinds of custom dictionaries that you can use: global and project:

  • Global Dictionary

    See Choosing the Location for Global Dictionaries.

  • Project Dictionary A project dictionary is just what it sounds like. It is unique to a particular project. When a project dictionary is created, it is stored in the Advanced folder in the Project Organizer.

    For example, you might choose to use this kind of dictionary if you are a writer with several clients, and each client has unique terms with unusual spellings. By using a project dictionary, you can keep those spellings tied to a certain project.

Thesaurus

The information that displays in the Thesaurus window pane comes from the dictionary you are using in your project. Keep in mind that only some of the available dictionaries also provide a thesaurus. If you are using a dictionary without a thesaurus, the Thesaurus window pane will be empty when you open it. See Using the Thesaurus.
