Associating Skins with Targets

After you add a skin to your project and edit the settings, you can associate it with a target. After you build the target, the output will be displayed in the skin. For HTML5 targets, you can also tell Flare not to use any skin at all.

how to associate a skin with a target
  1. Open the target from the Project Organizer.
  2. Do one of the following, depending on the type of output:
    • HTML5 Select the Skin tab.
    • Others Select the General tab.

  3. Click the drop-down arrow in the Skin field and select the skin that you want to associate with the target.

    If you have opened an HTML5 target and do not want to use a full skin at all, you can instead select none. If you do this, you might want to insert various kinds of proxies and use related skin components in order to provide navigation for your users. See About Proxies.

 Note

If you are producing HTML5 output, you can also associate skin components with a target, although it is more common to associate them with proxies. See Using the Menu Proxy, Using the Search Bar Proxy, Using the Search Results Proxy, Inserting Topic Toolbar Proxies, and Associating Skin Components with Targets.
