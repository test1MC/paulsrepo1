Associating Skin Components with Targets

See About Skins and About Proxies.

how to associate a skin component with a target
  1. Open an HTML5 target.
  2. Select the Skin tab.

  3. In the Component Default Skins section, click the drop-down arrow in any of the fields (Menu, Topic Toolbar, Search Bar, Search Results, Favicon) to select the skin component that you want to associate with the target.
