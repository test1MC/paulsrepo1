Generating Web-Safe Images

how to generate web-safe images
  1. Open a target.
  2. Select the Advanced tab.
  3. Select Generate "web-safe" images.
  4. From the drop-down list, select the image format that you want to use.
 note

If you are using this option, images that link to external locations (such as websites), rather than to images within the project, are not affected.
