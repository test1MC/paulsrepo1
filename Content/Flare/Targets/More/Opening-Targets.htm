Opening Targets

When you want to modify an existing target, use the following steps to open it. You can use the first set of steps to quickly open the primary target, or you can use the second set of steps to open a specific target (not necessarily the "primary" target).

how to open the primary target

  • Ribbon In the Primary Files section select Primary Target.
  • Keyboard Shortcut Press CTRL+F9.
 how to open a Specific Target from the Quick Launch Bar

how to open a specific target from the Project Organizer
  2. Double-click the Targets folder. The targets in your project are displayed.
  3. Double-click the target that you want to open. The Target Editor opens to the right, with the target page shown.
