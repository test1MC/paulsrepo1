example steps for using global project linking

Following are steps that you can perform when it comes to Global Project Linking.

  1. Import Project Files The Import Flare Project Wizard guides you through the steps for importing Flare files from another project. When you use this wizard, an import file is created automatically and placed in the Imports folder in the Project Organizer. See Global Project Linking—Importing Files from Other Projects.

    As an alternative to using the Import Flare Project Wizard, you can add an import project file manually (see Adding Project Import Files). Then you can use the Project Import Editor to edit settings and perform the import.

  2. (Optional) Re-import Project Files If you do not use the auto-reimport feature in your settings, you may want to manually re-import files from time to time. See Global Project Linking—Re-importing Files from Other Projects.
