Renaming Flare Projects in Windows

After creating a Flare project, you can rename it in Windows by changing the FLPRJ file name.

how to rename flare projects in windows
  1. Close the project if it is open.
  2. Open Windows Explorer.
  3. Locate the folder where you created the project.
  4. At the root of the main folder, you will see a file with an .flprj extension.
  5. Enter a different file name, without changing the extension.
