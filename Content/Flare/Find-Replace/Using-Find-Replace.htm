Using the Find and Replace in Files Feature

how to open the find and replace in files window pane

You do not need to have any files open in order to use this window pane to search for text.

  • Ribbon In the Find and Replace section select Find and Replace in Files.

  • Keyboard Shortcut Press CTRL+SHIFT+F.
 using the find and replace in files window pane using the find results window pane Adding Topic Results to a TOC or Browse Sequence

Complete the following steps if you want to add topics found in the results to a table of contents (TOC) or browse sequence:

  1. Open the Find and Replace in Files window pane, enter search criteria, and click Find All. The results are populated in the Find Results window pane at the bottom of the interface.

  2. Open the TOC or browse sequence.

  3. In the Find Results window pane, select the topic file(s) you want to add to the TOC or browse sequence.

  4. Drag the file(s) to the TOC or browse sequence and drop them at the location you want.

    By default, the text will be blue to indicate that a variable based on the title of the topic is being used. However, you can override the variable by double-clicking the TOC or browse sequence node and replacing the label with your own text. For more information, see Blue Text=Links to System Variables.
