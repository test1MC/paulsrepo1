About Bookmarks

Following are the primary bookmark tasks in Flare:

  • Inserting The main task involved with bookmarks is inserting them. See Inserting Bookmarks.
  • Renaming If you want to rename one of the bookmarks that you have inserted in a , you can easily change the name. See Renaming Bookmarks.
  • Deleting If you inserted a bookmark and no longer need it, you can permanently remove it. See Deleting Bookmarks.
  • Hiding and Showing Names When you insert a bookmark into a , both the bookmark icon and the bookmark name display on screen. You can show or hide the bookmark name from your screen display. When hidden, the screen displays shows only the bookmark icon at the insertion point. See Hiding and Showing Bookmark Names.
  • Viewing Broken Bookmarks See Viewing Broken Bookmarks.
  • Viewing Broken Links   See Viewing Broken Links.
  • Viewing Unused Bookmarks See Viewing Unused Bookmarks.
  • Viewing Used Bookmarks See Viewing Used Bookmark Links.
