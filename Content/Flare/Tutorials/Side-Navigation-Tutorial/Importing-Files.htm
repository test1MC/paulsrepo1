  10. In the local toolbar, click Import.

  11. Remember, we do not want to import all of the files, but rather only a few of them. So at the bottom of this dialog, click Clear All to remove all of the check marks.

  12. Click the check boxes for the following files:

    • Home.htm

    • Hero-Background.jpg We already have one in the Legacy project, but we will replace that with this newer image. In your actual project, you might decide to keep your own image or create a new one.

    • icon-checkbox.png

      icon-gears.png

    • icon-download.png

      icon-forum.png

      icon-pdfs.png

      Icon-Popular-Topics.png

      icon-videos.png

    • minus.png

      plus.png

    • Home-Page.flmsp
    • Other-Topics.flmsp

    • Styles.css

    • Alternate-Row-Color.css
    • Side-Navigation.flskn This is the main HTML5 Side Navigation skin that provides the look for the main menu and search bar, as well as the side flyout menu when the output is displayed on smaller devices. After this skin is imported, we will use it instead of our older Top Navigation skin.

    Use caution when selecting files to import. As mentioned, you likely won't have many files with the same names in your existing project, but you want to avoid overwriting any of your files that you want to keep.

  13. At the bottom of the dialog, click Accept.
  15. Click the x on the tab of the Project Import Editor to close it.

  16. In the Project Organizer, click the Import from Temp file and delete it.
