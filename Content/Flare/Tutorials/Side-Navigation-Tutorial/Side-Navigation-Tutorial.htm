Side Navigation Conversion Tutorial

Introduction

Welcome to the Flare Side Navigation Tutorial.

The Side Navigation project template in Flare contains several starter topics and other files intended for online HTML5 output that looks like a modern website with navigation on the side (left or right). It offers a frameless, modern alternative to the more traditional tripane format. Responsive output is automatically enabled with this type of output. This means the content display changes automatically depending on the width of the screen.
