Reorganizing the TOC

Reorganize the TOC in the Legacy Project
  2. In the Project Organizer, expand the TOCs folder.

  3. Double-click TOC (Master), the name of the TOC we will use for the Top Navigation output. The TOC opens to the right.

    The blue text indicates that the TOC entries are using system variables to keep the labels in sync with the titles of the topics. However, you can override any of the labels, typing whatever you want, in which the text will display in black.

  4. Most of the TOC items are relatively short, but we can edit the first one to give us a little more space in the top navigation menu. First, click on the item labeled Getting Started. Press F2 on your keyboard. Then replace the variable syntax with the word Get Started as the new label and press Enter.

    There are only six first-level items in our TOC, and that should look fine in the new output. But in your actual project, you might need to rearrange the TOC to reduce the number of first-level items, moving some items into others. The longer the TOC in your actual project, the more work you will need to do to reorganize.
