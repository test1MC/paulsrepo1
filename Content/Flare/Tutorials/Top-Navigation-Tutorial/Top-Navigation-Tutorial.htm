Top Navigation Conversion Tutorial

Introduction

Welcome to the Flare Top Navigation Tutorial.

The Top Navigation project template in Flare contains several starter topics and other files intended for online HTML5 output that looks like a modern website with navigation on the top. It offers a frameless, modern alternative to the more traditional tripane format. Responsive output is automatically enabled with this type of output.
