  2. The Add File dialog opens.

  5. Under the Source Project field, click Browse.

  7. Click Open. The path to the project file is loaded into the Source Project field in the Project Import Editor.

  8. In the Include Files field, click the down arrow and select All Files (*).

  10. In the local toolbar, click Import.

  11. Remember, we do not want to import all of the files, but rather only a few of them. So at the bottom of this dialog, click Clear All to remove all of the check marks.

  12. Click the check boxes for the following files:

    • Home-Page.flmsp
    • Other-Topics.flmsp
    • Styles.css
    • Alternate-Row-Color.css
    • Home-Searchbar.flskn This is a special skin component that the Temp project uses for the central search bar on the Home page. It's already designed the way we want, so we will bring it into our Legacy project.
    • Top-Navigation.flskn This is the main HTML5 Top Navigation skin that provides the look for the top menu and search bar, as well as the side flyout menu when the output is displayed on smaller devices. After this skin is imported, we will use it instead of our older Tripane skin.
    • Topics-Menu.flskn This is a special skin component used for the context-sensitive menu that appears next to each topic when it is opened. It is already modified with the look we want, so we will import this too.
    • Topics-Searchbar.flskn This is a special skin component used for the topic search bar that appears above the context-sensitive menu in each topic. It is already modified with the look and alignment we want, so we will choose this file as well.

    Use caution when selecting files to import. As mentioned, you likely won't have many files with the same names in your existing project, but you want to avoid overwriting any of your files that you want to keep.

  13. At the bottom of the dialog, click Accept.
  15. Click the x on the tab of the Project Import Editor to close it.

  16. In the Project Organizer, click the Import from Temp file and delete it.
