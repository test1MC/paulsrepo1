Building Output

After your targets are prepared, you are ready to build your final output. In this tutorial, you will build PDF and HTML output.

A PDF target has been created for you as part of the San Diego template. However, if you want to build HTML5 output, you must first follow the tutorial steps to create an HTML5 target. See Creating a New Target.

The following are covered in this tutorial:

  1. HTML5 Output If you want to create online output from your project, you can build HTML5 output. This output uses a modern layout with a top navigation similar to a website. See Building HTML5 Output.

    Note

    An HTML5 target has not been created for you. If you want to build this output, you must follow the tutorial steps to create an HTML5 target. See Creating a New Target.
  2. PDF Output If you want to create printed output from your project, you can build PDF output. A PDF target has already been created for you. See Building PDF Output.
