Adding Content and Features to a Project

Now that you have created a project, it is time to add content to it. Adding content gives your project some substance. You can add a variety of different features to a project.

In this tutorial, you will add the following:

  1. Topics You might say that topics are the most important part of a Flare project because they hold and display your content. See Creating Topics.
  2. Images You can insert images into topics. Flare supports most common image file formats. See Inserting Images.
  3. Cross-References These are an easy way to navigate from one topic to another. See Inserting Cross-References.
  4. Drop-Down Text This is a good way to segment a topic and make it easier for users to find and open specific content in it. See Inserting Drop-Down Text.
  5. Snippets These are chunks of content that you can insert into multiple topics so that you don't have to keep copying and pasting, or re-adding the content. When you need to make a change to the content in the snippet, you only need to change it once; that change appears anywhere you used the snippet. See Using Snippets.
  6. Tables You can use tables for many reasons, such as comparing elements. See Inserting Tables.
  7. Variables These are small bits of content that you can use over and over in your project. Unlike snippets, which are longer chunks of content, variables are usually a word, short term, or a number (e.g., your company name or phone number, or a date). Variables are stored in variable set files. See Using Variables.
