Applying Styles

Now we are going to apply styles to content in our topics. In addition to previously existing styles, we will also apply the new style class you created.

Apply Styles to the Front2 Topic

Apply Styles to the Front2 Topic

  1. Select the Home ribbon, and in the Styles section click Style Window. The window pane opens on the right side of the interface.

  2. Open the Front2.htm topic.

  3. The heading style is already applied, so we can ignore that. Click somewhere in the first paragraph below the heading. The Styles window populates with relevant styles.

  4. In the Styles window pane, select p.large-new, the style class you created.

    In the topic, the paragraph is formatted, increasing in size. This pushes the second-level heading text to the third panel, which is using the second body frame.

  5. Select the short copyright paragraphs at the bottom of the first panel.

  6. In the Styles window pane, select p.copyright.

    In the topic, the paragraphs are formatted accordingly, now using a smaller font.

  7. Click the first text at the top of the third panel.

  8. In the Styles window, select the h2 style.

    This applies a different look to that text. Also, the h2 style is set to automatically use a frame break before it, which ensures it begins at the beginning of the body frame, rather than having other text above it.

  9. The three multi-line paragraphs in the third panel are already using the p style, which is what we want. But we need to apply styles to the two other second-level headings. First, click in the next second-level heading, and in the Styles window select h2.no-frame-break.

    The style changes the look of the text, but does not push it to the next panel, because this particular style is not set with a frame break.

  10. Select the next second-level heading, and once again click the h2.no-frame-break style class.

    The look of the text changes. Also, the content below it moves automatically to the fourth panel because there is no room for it on the third panel anymore.

  11. In the fourth panel, click to the right or left of the logo image. Do not select the image itself, because we want to apply a style to the paragraph holding the image. Then in the Styles window, select the p.logo style class.

    The logo is now centered in the panel.

  12. Click to the right or left of the variable below the logo (do not select the variable itself). Then in the Styles window, select the p.title style class.

    The look of that content changes.

Apply Styles to the Back2 Topic

Apply Styles to the Back2 Topic

In the other topic, we are going to apply styles that already exist in the stylesheet.

  1. Open the Back2.htm topic.

  2. Use the same steps above to apply the following styles to the paragraphs in the first panel of the Back2 topic:

    • Get Started in Apply the p.get-started1 style class.
    • Just a Few Steps Apply the p.get-started2 style class.
    • For more help or troubleshooting… Apply the p.large-light style class.
    • www.yourcompany.com/help Apply the p.large-light-url style class.
    • Created Using MadCap Flare Apply the p.created-using-flare style class.

  3. In the second panel, click on the Step 1 text, and in the Styles window select the p.step style class.

    The look of the text changes.
