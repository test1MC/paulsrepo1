Creating Page Layouts

Page layouts are used to configure the pages for print-based output and are especially important in this particular template because of its unique characteristics (e.g., multiple pages, background images). This template comprises only two topics—one for the front of the product foldout and the other for the back of the foldout. Therefore two page layouts are required—one to be associated with each topic.

When you go through the following steps, you will see that two page layouts already exist in the project, and they're already completely edited. If you want, you can open these files and take a look around to see what you will be producing in the end. You can also build one of the targets in the project to see how these page layouts are incorporated into the output.

In the following sets of steps, we will create two new page layouts, replicating the look and substance of the existing page layouts.

Front Page Layout

front page layout

The existing front page layout in the project is simply named "Front." We'll give our new page layout a slightly different name.

Create a Page Layout File

  1. In the Content Explorer, expand the Resources folder, then expand the PageLayouts folder. Notice the existing two page layouts (Back and Front).

  2. Right-click the PageLayouts folder, and from the context menu select New > Page Layout.

    The Add File dialog opens.

  3. Under the Factory Templates, select the Default template, because it is quite simple and requires fewer changes than other templates.

  4. In the File Name field at the bottom of the dialog, type Front2 as the name for the new page layout.

  5. Click Add. The page layout is added to the Content Explorer, and the Page Layout Editor opens to the right, displaying the page layout.

  6. Before we continue with the rest of the steps for editing the page layout, let's make a couple of quick changes in it.

  7. Click the header frame (the rectangle) at the top of the page layout.

  8. Press Delete on your keyboard.

  9. Scroll down to the bottom of the page layout. You will notice the footer frame, which is automatically selected.

  10. Press Delete to remove it.

For more information about adding page layout files, see the online Help.

For more information about adding page layout files, see Adding Page Layouts.

 Set page size and margins

Now that we have created our first page layout, we will start making changes to it. To begin, we will adjust the page size and margins.

  1. In the local toolbar of the Page Layout Editor, click (click the face of the button, not the down arrow).

    The Page Properties dialog opens.

  2. Select the General tab.
  3. Click the Page Size drop-down and select (custom).
  4. In the Custom Width field, enter 20, and for the unit of measurement select Inch (in).
  5. In the Custom Height field, enter 5, and for the unit of measurement select Inch (in).
  6. Keep the Orientation field set to Portrait.

  7. In all of the Margins fields, enter 0.

    You can set some value for the margins on your page if you want. However, in our template we chose to set them all to 0 and specify the margins on our CSS styles instead.

  10. Because the view of the page layout is so large, it is difficult to see what's going on so far. Therefore, in the local toolbar, click View and select Fit Width.

    The page layout is zoomed out so you can see all of it.

For more information about specifying the page size and margins, see the online Help.

For more information about specifying the page size and margins, see Specifying Page Size and Orientation and Specifying Page Margins.

 Set Background Image

The front page layout is almost finished, but we want to add a background image to the first body frame, and position it on the right side. We’ve already set the margins in that frame so that the text is shown to the left of it, with no text displaying on top of the image. But if we wanted, we could have text or other content display above the background image, which could then function more as a watermark.

You can also add a background image to an entire page layout, and not just a body frame.

  1. Double-click the first body frame. The Frame Properties dialog opens.
  2. Select the Background tab.
  3. In the Image section, click Browse.

  4. Under the Resources > Images folder, select Background-Panel2.png. Click OK when you are done.

  5. In the Repeat field, select No repeat. Also, set the X field to Right.

For more information about backgrounds in page layouts, see the online Help.

For more information about backgrounds in page layouts, see Specifying Page Background Settings.
