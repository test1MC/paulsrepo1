Link to Browse Sequence Dialog

This dialog is used to link entries in a browse sequence to another browse sequence or link entries in a table of contents (TOC) to a browse sequence. For more information see About Browse Sequences.

  1. Open a browse sequence or TOC.
  2. Select an entry and click .
  3. In the Properties dialog, click the Select Browse Sequence button.

Button/Section

Description

Toggles between showing or hiding the subfolders holding the browse sequence files.

Splits the dialog into two halves. The right half shows browse sequence files. The left side shows the subfolder holding the files.

Shows or hides hidden files and folders.

Moves up one folder.
