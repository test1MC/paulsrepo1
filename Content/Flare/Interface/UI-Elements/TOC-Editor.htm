This button toggles between Tree View and Grid View when you click it. Tree View is the traditional format of the , displaying books and items in a hierarchy. Grid View displays all of the books and items in a column format. In either view you can select a range of entries and set the same property information for all of those items.

This option is enabled in Grid View only.

Expands all of the books so that you can see the items under each book (enabled in Tree View only).

Collapses all of the books so that you can no longer see the items under each book (enabled in Tree View only). You can only see the books.

If you have created and applied condition tags, clicking this button toggles between showing and hiding them in the Editor. This option is enabled in Tree View only.

Click this button to create a new topic and simultaneously link it to a new item in the . When you click this button, it opens the Add File dialog, which you use to create the new topic, associate it with a topic template, and give it any name you want. When you are finished with this dialog, the new topic is created and it opens in the XML Editor. You can click the tab containing your to return to the Editor. You will notice that the item is created at the selected location in the . For example, if you have a book selected and then click this button, the new item will be added under the book.

Click this button to add a new item to the . You can link the item to a topic, a TOC, a browse sequence, another Flare project, an external Help system, or a MadCap Mimic movie. The item is created at the selected location in the . For example, if you have a book selected and then click this button, the new item will be added under the book. (As an alternative, you can click existing topics in the Content Explorer and drag them to the Editor. Doing this creates a new item with the same name and automatically links it to the topic.)

Click this button if you want to toggle the behavior of double-clicking an item in the . The default behavior (without the toggle button selected) is that double-clicking an item will open the properties dialog, and holding down the CTRL key while double-clicking will open the file linked to the item. If you activate the toggle button, this behavior is reversed; double-clicking an item opens the file linked to it, and holding the CTRL key while double-clicking will open the properties dialog.
