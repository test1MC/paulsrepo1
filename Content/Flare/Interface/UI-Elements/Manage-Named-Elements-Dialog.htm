Manage Named Elements Dialog

This dialog opens whenever you attempt to add or rename a named element in the XML Editor. You would do this when you want to add a toggler to your content. See Inserting Togglers.

Button/Section

Description

New name

Enter the name of the element in the field. You can also rename the element by entering a new name in the field.

Named Elements

All of the available named elements in this topic or snippet.

Tag

The tag for each element (e.g., <p>, <li>).

Text

The text of the element.

Go To

Click this button to quickly jump to the selected element in the topic or snippet.
