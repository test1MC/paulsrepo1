Mode Warning Dialog

This dialog opens if you have enabled Index Entry Mode or Concept Entry Mode in a previous session, but did not turn it off when you exited . From this dialog, you can elect to continue in that mode as you work in your project or turn it off.

For more information about these modes, see Inserting Index Keywords and Inserting Concepts into Topics.

note

You can disable these warning messages from the Index & Concepts tab in the Options dialog .

Button/Section

Description

Action

Select one of the options to indicate the immediate action to be taken after the dialog is closed.

  • Continue in mode The Index Entry Mode or Concept Entry Mode will continue to be enabled after this dialog is closed. You can turn off the mode manually by right-clicking in the topic and selecting the mode.
  • Switch to normal selection mode The mode will be turned off so that you can edit topics without index keywords or concepts automatically being created.

Options

Select one of the options to indicate how to handle future occurrences when the Index Entry Mode or Concept Entry Mode is turned on when close and then reopen .

  • Check mode and display this dialog when first topic opened You can avoid seeing this dialog in the future by deselecting this option. However, this does not mean the mode will automatically be turned off for you. It simply means that you will not be warned.
  • Auto-switch to normal selection mode on launch Select this option if always want to disable the mode automatically when you open the program in the future. You can always turn the mode on again manually.
  • Continue in mode The Index Entry Mode or Concept Entry Mode will continue to be enabled. You can turn off the mode manually by right-clicking in the topic and selecting the mode.
