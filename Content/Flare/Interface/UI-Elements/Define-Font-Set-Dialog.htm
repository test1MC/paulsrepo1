Define Font Set Dialog

This dialog is used add fonts to and remove fonts from font sets. You can also rearrange fonts in a font set to change their order.

  • Creating Font Sets You can create new font sets. See Creating Font Sets
  • Editing Font Sets You can edit existing font sets. See Editing Font Sets.

Button/Section

Description

Name

Enter a name for the font family. If you are editing the font set, this name cannot be changed.

[Font Family field]

This is the field above the list of fonts on the left side of the dialog. If you know the name of the font family you want to add to your font set, you can enter it in this field.

[Available Font Families List]

This is the list of fonts on the left side of the dialog. You can add these fonts to the font set.

[Selected Font Families List]

This is the list of fonts on the right side of the dialog. These fonts are part of the font set.
