how to open the concepts window pane

  • Ribbon In the Reference section select Concept Window.
  • Keyboard Shortcut Press Shift+F9.
 what you can do in the concept window pane

Following are the primary tasks that you can perform in this window pane.

  • Create Concepts There are two ways to link a topic to a concept using this window pane. First, you can open the topic, place your cursor where you want to insert the concept, and use the top area of the Concept window pane to create the keyword (see Inserting Concepts). Second, you can also use the bottom area of the Concept window pane to add concepts (see Adding Concepts) and assign topics to them (see Assigning Topics to Concepts).
  • Remove See Deleting Concepts.
  • Rename See Editing Concepts.
