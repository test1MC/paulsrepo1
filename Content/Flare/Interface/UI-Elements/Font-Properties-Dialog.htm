Font Properties Dialog

This dialog is used to affect the look and behavior of text in your content in various ways. These settings can be applied locally (i.e., directly) to the text or by applying a style to selected text. Modifying the style is typically preferable to changing the settings locally. Different settings are available depending on the user interface you are using to apply the font properties (e.g., the dialog may have an Advanced tab or it may not, depending on where you open it).

How to Open the Font Properties Dialog

    • Ribbon At the bottom of the Font section click .

    • Keyboard Shortcut Press CTRL+SHIFT+B.

    The Font Properties dialog opens.

 What You Can Do in the Font Properties Dialog

Following are the primary tasks that you can perform in the Font Properties dialog:

  • Create Font Sets You can create font sets where you can specify alternate fonts to use in your content. If the first font in the set is not available, the system uses the second font; if the second font is also not available, the system uses the third font; and so on. Click Font Sets to use this feature. See Creating Font Sets.
  • Delete Font Sets If you do not need a font set anymore, you can delete it from the Font Set Manager. See Deleting Font Sets.
  • Edit Font Properties You can select font styles and effects to change the appearance of text. See Ways to Edit Font Properties.
  • Edit Font Sets You can edit font sets after you create them if you want to add or remove fonts from the font set. You can also change the order of fonts in the set if you want to give a different font a higher or lower priority in the font set. Click Font Sets to use this feature. See Editing Font Sets.
  • Group Fonts The font list contains groups for pinned fonts, recently used fonts, font sets, and all system fonts. The font list contains groups for all fonts, recently used fonts, and all system fonts. You can expand or collapse each group to limit the list of fonts to those that you use the most and hide from view those that you do not use. See Using Font Groups.
  • Pin Fonts and Font Sets You can pin your favorite fonts and font sets to the top of the font list so you can easily find them again later. See Pinning Fonts.
  • Preview Font Properties As you select different font styles and effects, the Preview bar at the bottom of the Font Properties dialog shows you how your text will appear in your document.
 Ways to Edit Font Properties

Following are some of the more common ways that you can edit font properties.

  • Case Read more… See Setting the Case of Text.
  • Color of Background Read more… You can set a background color for text. See Setting the Background Color for Text.
  • Color of Text Read more… You can select a color for text. See Setting the Color for Text.
  • Font Family Read more… You can select a font family (e.g., Arial, Verdana) for text to change its look. See Selecting Fonts.
  • Italics Read more… See Setting Italic Text.
  • Italic Correction Read more… See Setting Italic Correction.
  • Letter Spacing Read more… See Setting Letter Spacing.
  • Overline Read more… You can specify that a horizontal line should be placed above text (opposite of underlined text). See Setting an Overline on Text.
  • Redacted Text Read more… See Setting Redacted Text.
  • Size  Read more… You can specify the size of text, choosing from many different units of measurement. See Setting the Font Size.
  • Smallcaps Read more… You can specify that text should be displayed in all capital letters, but in a small size. See Setting Smallcaps on Text.
  • Strikethrough  Read more… You can specify that a horizontal line should be added through the middle of the text, providing the effect of content that has been "striked through." See Setting a Strikethrough Effect on Text.
  • Subscript Read more… See Setting Subscript Text.
  • Superscript Read more… See Setting Superscript Text.
  • Underline Read more… You can easily underline text. See Setting Underlined Text.
  • Word Spacing Read more… See Setting Word Spacing.
