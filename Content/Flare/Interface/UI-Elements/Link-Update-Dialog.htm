Link Update Dialog

This dialog gives you the opportunity to update any links to a file when you attempt to make changes to it or delete it.

There are numerous ways to open this dialog. For example, let's say there are hyperlinks in other topics that point to the current topic that you are working on. If you try to rename the topic, this dialog opens because it detects those links. You then have the opportunity to update the links or not to update them.

Button/Section

Description

File

Displays the name of each file containing a link to the file in question.

Path

Displays the location of each file in the project.

Update Links

Click this button if you want to update all of the links in other files to the file in question.

Do Not Update Links

Click this button if you do not want to update all of the links in other files to the file in question. If you select this option, you must manually fix or remove the links in those files. Otherwise, you will experience errors when you generate the output.
