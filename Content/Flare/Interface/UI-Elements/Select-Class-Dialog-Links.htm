Select Class or ID Dialog (Links)

This dialog is used to apply a CSS style or a style ID to a link (such as a text hyperlink). For hyperlinks, you can create classes of the "a" style in the Stylesheet Editor; those classes will then become available in the Select Class or ID dialog. Generic classes and IDs are also available for selection in this dialog (see Generic Classes and Identifiers).
