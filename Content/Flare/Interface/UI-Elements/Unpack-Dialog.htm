Unpack Dialog

This dialog is used to open a project that has been zipped (compressed into a single file). Anyone who also uses can open and unzip the file, retuning the project to its normal state so that it can be edited. See Opening Zipped Projects, Zipping Projects, and Zipping and Emailing Projects.

  • Double-click the zipped FLPRJZIP file.

Button/Section

Description

Package to Unpack

Displays the name of the zipped file and the path to the folder where it is located.

Project Name

Displays the name of the project.

Unpack To Folder

Displays the location where the project files will be placed after the compressed file has been unzipped.

Browse

Click this button to choose a folder where you want to place the unzipped project files. After you select a location, the "Unpack To Folder" fields is populated.
