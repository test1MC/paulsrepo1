Reviewed Topic Editor

This editor can be used to view a reviewed topic. From this editor, you can also accept a topic from it so that it is copied to the source file in the project. See About Reviews and Contributions.

  1. In the Packages section select File Reviews. The File Reviews window pane opens.
  2. Make sure Inbox is selected at the top of the window pane. All reviewed files that you have brought into your project, but have not yet been accepted, are listed in the pane.
  3. Double-click one of the rows containing a topic file.

Button/Section

Description

Opens the source topic in the XML Editor.

Lets you accept the reviewed topic. After accepting the topic, it is converted to an HTM file (just like your regular topics) and overwrites the source topic in your project. Any edits or annotations (comments) in the reviewed topic therefore become part of the original topic. See Accepting Reviewed Files.
