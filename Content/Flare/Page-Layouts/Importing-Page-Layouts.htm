Importing Page Layouts

how to import a page layout

  1. In the Content section select New > Page Layout.

    The Add File dialog opens.

  3. In the Open dialog, navigate to the FLPGL file that you want to import. Then select it and click Open. The Source File field now contains the path to the file that you are importing. Also, the name of the file is displayed in the File Name field.
  4. In the File name field, type a new name for the page layout.
  5. (Optional)   See Applying Conditions to Content.
  6. (Optional)   See Associating Tags with Files.
  7. Click Add. The page layout is added to the Resources\PageLayouts subfolder in the Content Explorer. The Page Layout Editor opens, displaying the page layout.
