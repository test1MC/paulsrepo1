Associating Master Page Layouts with Projects

See About Page Layouts.

The following steps show how to associate a "master" page layout at the project level.

how to associate a master page layout with a project
  1. In the Properties section select Project Properties.
  2. Select the Defaults tab.
  3. Click in the Master Page Layout field, and from the drop-down, select the page layout that you want to use for all topics in the project.
