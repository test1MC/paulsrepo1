Automerging Browse Sequences

The following automerge tasks can be performed on browse sequences:

  • Merging Output for HTML5 You can merge your master project's browse sequence for HTML5 output. See Runtime Merging Server-based HTML5 Output.
  • Merging Output for WebHelpPlus You can merge your master project's browse sequence for WebHelp output. See Runtime Merging Output Using WebHelp Plus.
