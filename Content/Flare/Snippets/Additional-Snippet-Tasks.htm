Additional Snippet Tasks

You can perform the following additional tasks with snippets:

  • Converting Snippets to Text After you create or add a new snippet to your project, you may decide that you want the snippet to be regular text instead. See Converting Snippets to Text.
  • Deleting Snippets You can delete a snippet from an existing topic. See Deleting Snippets.
  • Importing Snippets You can import a snippet from another project into your current project. See Importing Snippets.
  • Inserting Page Numbers You can insert page numbers, which are actually snippets, into your frames as a placeholder for your output. See Inserting Page Numbers into Frames.
  • Pinning Snippets You can pin your favorite snippets to the top of the list so you can easily find them again later. See Pinning Snippets.
  • Replace Snippets After you create or add a new snippet to your project, you can insert it into any of your topics. Later, if you decide that a different snippet is a better fit for the topic, you can select a new snippet to replace it. See Replacing Snippets.
  • Using Snippet Groups You can select snippets from groups of pinned snippets and recently used snippets. You can expand or collapse each group. Snippet groups are a way to quickly access the snippets that you use the most. See Using Snippet Groups.
