Creating Font Sets

Because different types of computers and operating systems do not always have the same fonts, it is recommended that you try to include fonts that cover all of the bases.

example

Example

Let's say you want to use Arial if possible. Therefore, you list that font type first in the font set. Next you might add a similar font that is found on Macintosh computers, such as Helvetica, since Macs do not use Arial. And third, you might add a generic font type, such as Sans-serif.

 how to create a font set
  1. (Optional) You only need to do this if you are opening the Font Set Manager from a location other than the Stylesheet Editor.
    • Stylesheet Editor In the local toolbar of the Stylesheet Editor, click and select Manage Font Sets.

    • Ribbon At the bottom of the Font section click . The Font Properties dialog opens. Click Font Sets.

    • Keyboard Shortcut Press CTRL+SHIFT+B. The Font Properties dialog opens. Click Font Sets.

    The Font Set Manager opens.

  3. In the Font Set Manager, click . The Define Font Set dialog opens.
  4. In the Name field, enter a name for the new font set.
  5. From the list on the left side of the dialog, double-click each font family that you want to add to the set, or select a font family and click to add it to the set. Fonts you have selected appear in the list on the right side of the dialog.
  6. (Optional) If you want to reorder the font families in your font set, select a font family from the list on the right side of the dialog, then use the and buttons to rearrange their order. When you are finished, the first font family should be the preferred font family that you want to be used, then the second, then the third, and so on.
  7. (Optional) If you want to remove a font family from the list of selected font families, select it and click .
  9. Click OK in the Font Set Manager dialog.
  10. Click OK in the Font Properties dialog.

 What's Next?

After you create a font set, you can select it when applying fonts to content. You can also select that font set from the Defined Font Sets group in the Font Family field in the Font section of the Home ribbon. See Selecting Fonts.
