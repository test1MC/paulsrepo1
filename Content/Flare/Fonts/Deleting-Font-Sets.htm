Deleting Font Sets

A font set is just what it sounds like—a collection of fonts or font families. If you do not need a font set anymore, you can delete it from the Font Set Manager.

how to Delete a font set

    • Ribbon At the bottom of the Font section click . The Font Properties dialog opens. Click Font Sets.

    • Keyboard Shortcut Press CTRL+SHIFT+B. The Font Properties dialog opens. Click Font Sets.
    • Stylesheet Editor In the local toolbar of the Stylesheet Editor, click and select Manage Font Sets.
