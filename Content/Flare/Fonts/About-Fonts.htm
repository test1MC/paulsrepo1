Following are some ways that you can work with fonts in :

  • Selecting Fonts See Selecting Fonts.
  • Editing Font Properties See Editing Font Properties.
  • Embedding Fonts You can embed fonts into HTML5 output. This is particularly beneficial when using custom fonts or other fonts that may not be installed on end users' machines. This method also works with font formats that are not natively supported in Flare (e.g., .woff, .eot). See Embedding Fonts.
  • Creating Font Sets You can define and apply a font set. See Managing Font Sets.
  • Pinning Fonts and Font Sets See Pinning Fonts.
  • Font Groups See Using Font Groups.
