Moving Identifiers to Different Headers

In the Alias Editor, you can move identifiers to a different header file (if you have multiple header files already). If you do not yet have a header file, you first need to add one (see Adding Header Files).

how to move identifiers to a different header
  1. Open an alias file.
  2. Select the identifiers that you want to move to another header file.
  3. Right-click and from the context menu select Move Identifiers > [Name of Header File].
