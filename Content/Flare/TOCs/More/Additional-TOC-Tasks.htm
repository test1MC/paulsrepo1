Additional Table of Contents Tasks

In addition to the primary tasks you can perform related to tables of contents (TOCs), you can also do the following:

  • Creating a Binary TOC See Creating a Binary TOC.
  • Deleting You can delete a browse sequence in the Project Organizer. See Deleting Tables of Contents.
  • Finding and Fixing Issues See Finding and Fixing Issues in a Table of Contents.
  • Importing Not only can you add a new table of contents (TOC), but you can also import an existing TOC (FLTOC file). See Importing Tables of Contents.
  • Locating Topics See Locating Topics in TOCs.
  • Managing Mini-TOCs You can create mini-TOC proxies in your project, suppress page numbers for print output, and apply styles on your mini-TOCs. See Managing Mini TOCs.
  • Managing Print TOCs You can auto-generate TOCs, create headings, apply styles, and other tasks on your Print TOCs. See Managing Print TOCs.
  • Opening You can quickly open a TOC anytime you need to work on it. See Opening a TOC.
  • Renaming See Renaming Tables of Contents.
  • Synchronizing Navigation Elements with TOC Entries See Synchronizing Navigation Elements with TOC Entries.
  • Synchronizing with Open Topics See Synchronizing the TOC with Open Topics.
  • Viewing Topics Not in a TOC See Viewing Topics Not in a TOC.
