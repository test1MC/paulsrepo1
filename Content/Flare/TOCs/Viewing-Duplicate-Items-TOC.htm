Viewing Duplicate Items in a TOC

You can view a list of all files in your project that have been added more than once to a TOC using the Duplicate TOC Items window pane.

how to view duplicate items in the toc
  1. Open a project.
  2. In the Dynamic Reports section, select More Reports > Duplicate TOC Items. The Duplicate TOC Items window pane opens, displaying all files that have been added more than once to a TOC.
    • File
    • Title
    • Folder
    • TOC Displays the name of the TOC where the multiple links to the file are found.
    • Count Displays the count.
  5. You can open any file in the list to view or modify it in an editor.
