Auto-generating a TOC

how to create a table of contents automatically note

For print-based outputs, you can also use an option to auto-generate TOCs for the output. See Auto-generating TOCs for Print.

 What's Next?

After you create a TOC, you need to enable TOCs in the skin you want to use for the target. This applies to all types of skins, except for HTML5 Side and Top Navigation skins. See Enabling Tables of Contents in Skins.
