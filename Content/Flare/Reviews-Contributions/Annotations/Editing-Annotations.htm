Editing Annotations

See Inserting Annotations.

You can easily make changes to an annotation. The following steps show how to edit an annotation in Flare. If you have sent a file for review on MadCap Central, annotations can also be edited in the Central interface. For more, see About the Central Review Process and the Central online Help.

how to edit annotations
  1. In the Changes section select Show Changes.

  2. Click in the rectangle containing the annotation text and make your changes.

 notes

See Changing Your User Name and Initials.

You can modify the look of content in a topic to which an annotation points. For example, you might want annotated text to be displayed in the XML Editor with red font and a yellow background. This does not change the text as it will be shown in the output, but rather only as it is displayed in the XML Editor. To do this, you can modify the MadCap|annotation style in the Stylesheet Editor. See About Styles and Stylesheets and Editing Styles in a Regular Stylesheet.
