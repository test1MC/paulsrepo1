Accepting or Rejecting Tracked Changes (Central Reviews)

how to accept or reject tracked changes
  1. From the File Reviews window pane, open the topic or snippet.
  2. Make sure you have changes set to be displayed. See Hiding and Showing Changes and Annotations.

  3. You can perform any of the following tasks.

    accept all changes in the File accept single change

    reject all changes in the File reject single change

Notes

After reviewing annotations in a file, you can delete them if they are no longer needed. See Deleting Annotations.

 What's Next?

After you accept and reject changes in a reviewed file, you need to accept the file back into Flare. See Accepting Reviewed Files.
