Workflow
  1. Original Author—Invite Reviewer to Central License See the Central online Help.
  2. Reviewer—Set Password

  3. Original Author—Select File(s), Choose Reviewer(s), and Send

    See Sending Files for Review.

  4. Reviewer—Open Pending File and Edit/Annotate

    For details of the editor interface, see the Central online Help.

  5. Reviewer—Submit File See the Central online Help.

  6. Author—Close Review File See Closing Review Files.

  7. Author—Accept/Reject Changes See Accepting or Rejecting Tracked Changes.

  8. Author—Accept File See Accepting Reviewed Files.
