Viewing Used Variables

how to view used variables
  1. Open a project.

  2.  In the Dynamic Reports section, select Used Items > Used Variables.

    The Used Variables window pane opens, listing all of the variables used throughout the project, including snippet variables.

    • Variable
    • File
    • Title
    • Folder
    • Tag
  5. You can open any file in the list to view or modify it in an editor.
