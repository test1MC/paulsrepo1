Additional Variable Tasks

In addition to the primary tasks you can perform related to variables, you can also do the following:

  • Chapter, Section, and Volume Number Variables See Inserting Chapter, Section, and Volume Number Variables into Frames.
  • Converting to Text See Converting Variables to Text.
  • Date/Time Variables
    • Inserting Into Master Pages See Inserting Custom Date/Time Variables Into Master Pages.
    • Updating See Updating Manual Date/Time Variables.
  • Definitions
    • Overriding in Snippets See Overriding Variable Definitions in Snippets.
    • Overriding in Targets See Overriding Variable Definitions in Targets.
  • Deleting If necessary, you can delete a variable from the Variable Set Editor. See Deleting Variables.
  • Groups Variable groups are available for pinned variables and recently used variables. You can access variable groups from both the Variables dialog and the Insert Variable drop-down in the local toolbar of the XML Editor. See Using Variable Groups.
  • Heading Variables See About Heading Variables.
    • Inserting into Frames See Inserting Heading Variables Into Frames.
    • Inserting into Master Pages See Inserting Heading Variables Into Master Pages.
  • Names—Hiding and Showing See Hiding and Showing Variable Names.
  • Pinning You can pin your favorite variables so you can easily find them again later. Pinned variables appear in a group that you can expand or collapse. This is a way to limit the list of variables to those that you use the most. See Pinning Variables.
  • Running Head Variables See About Running Head Variables.
    • Inserting into Frames See Inserting Running Head Variables into Frames.
    • Inserting into Master Pages See Inserting Running Head Variables Into Master Pages
  • Variable Sets
    • Adding See Adding Variable Sets to Projects.
    • Importing See Importing Variable Sets.
    • Opening Variable Sets You can open a variable set that you have added or imported to your project. See Opening a Variable Set.
    • Renaming Variable Sets You can rename a variable set in your project. See Renaming Variable Sets.
