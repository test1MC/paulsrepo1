Viewing Variable Suggestions

how to view variable suggestions
  1. Open a project.

    • Ribbon  In the Dynamic Reports section, select Suggestions > Variable Suggestions.

    • Analysis Summary Window Pane Double-click the row that mentions variable suggestions.

    The Variable Suggestions window pane opens, listing all of the variable suggestions for the project.

    • Suggestion Displays the suggestion that Flare is making to you (i.e., "Replace with Variable").
    • Type Displays the name of the variable set and the name of the variable (e.g., Primary.CompanyName).
    • Context Displays the context of the variable in question. In other words, it displays words that are found before the variable tag.
    • File Displays the path and name of the file where the content in question is found.
