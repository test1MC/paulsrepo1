Editor Tools

Flare provides many tools to assist you when you are working in its editors, including macros, customizable shortcuts, structure bars, and more.

  • Auto Suggestion Read more… When you first start typing content, a popup may display below your text as you type. This is the Auto Suggestion feature, which analyzes text as you type and compares it to other content (e.g., phrases, variables, snippets) that already exists in the project. This means you can quickly select existing content from the popup so that you do not have to finish typing it. See About Auto Suggestion.
  • Characters and Symbols Read more… You can insert characters and symbols in content files. You can also create quick characters, as well as work with font types and smart quotes. See About Characters and Symbols.
  • Dictionary Read more… See About Dictionaries.
  • Drag and Drop Read more… You can easily move content (e.g., text, images, etc.) around in topics. One method is drag-and-drop. Simply highlight the content that you want to move, click and hold down the mouse button to select it. Then drag the content the new location in the file. You can turn this feature on and off as necessary. See Enabling and Disabling Drag and Drop.
  • Find and Replace Read more… Flare provides multiple features for finding and replacing content in your project. See About Find and Replace.
  • Macros Read more… See About Macros.
  • Shortcuts Read more… lets you work more quickly and efficiently by making use of keyboard shortcuts in a variety of ways. See About Shortcuts.
  • Spell Check Read more… You can use spell checking features, based on your built-in and custom dictionaries. See About Spell Check.
  • Structure Bars Read more… See About Structure Bars.
  • Text Analysis Read more… See Using Text Analysis.
  • Thesaurus Read more… See Using the Thesaurus.
  • Zoom/Scale Read more… The bottom part of the XML Editor includes a toolbar that lets you use zoom/scale features to better see your content. The features that are available depend on the layout mode you are working in (e.g., Web Layout, Print Layout). See Using the Zoom/Scale Feature.
