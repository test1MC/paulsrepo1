Key Features

key features Skins—Multiple Options Side Navigation Skin Top Navigation Skin Tripane Skin Skinless Output Skin Components Note Frameless

Pages in output do not use <frameset> or <frame> tags, like they do in regular output. In addition, HTML5 Side and Top Navigation outputs are completely frameless, meaning they also do not contain iframes. This means that output is much easier for web crawlers to find, thus improving search engine results.

If your HTML5 output is enabled with MadCap Pulse, the comments area at the bottom of topics is wrapped in an iframe. Therefore, the output is technically not entirely frameless. However, because it is only the comments that are contained in the iframe, the main topic content still retains the benefits (e.g., better search results) of frameless output.
