  4. Activate your Salesforce® license using the following steps.

    1. Click Activate.

    2. Enter the license key.

    4. Enter your information:
      • First Name Enter the first name of the user associated with the license key.
      • Last Name Enter the last name of the user associated with the license key.
      • Email Enter the email address associated with the license key.
    6. Select either Internet Activation or Manual Activation.
      • If you chose Manual Activation, follow the instructions on-screen to copy and paste the information in an email to customerservice@madcapsoftware.com.
    8. If you chose Internet Activation, a confirmation displays to indicate that your license has been activated. Click Finish.

  5. Click the Login Credentials button. You can then enter the user name and password required for publishing to Salesforce®.

    If you have created a sandbox environment in Salesforce®, select the Login to Sandbox option if you wish to login to it.

  6. Complete the MadCap Connect for Salesforce® fields.

  7. Enter or select an http address in the View URL field to be used for viewing the published files. This field is optional and is for your own internal purposes.

  8. Select the following upload options when you publish to Salesforce®The Upload Log File option is automatically selected for Salesforce®.

    • Upload Only Changed Files Select this option if you want to republish only the files that have changed. This can save significant time.
    • Remove Stale Files Select this option if you want to identify files that were previously published but are no longer needed. will then remove such files from the destination.
