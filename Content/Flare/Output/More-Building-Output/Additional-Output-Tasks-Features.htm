Additional Output Tasks and Features

In addition to the primary tasks and features related to output, you can also do the following:

  • Testing See Testing Output.
  • Viewing You can view the generated output for any of the targets in your project. See Viewing Output.
  • Opening Build Logs See Opening Build Logs.
  • Cleaning (Deleting) Output See Cleaning (Deleting) Output Files.
  • Creating Pre- Post-Build Events In the Target Editor, you can add pre- or post-build events to your target. This lets you run command line events before or after the target is generated. See Creating Pre- and Post-Build Events.
  • Ignoring Warnings See Ignoring Warnings When Building Output.
  • Improving Processing Performance You can improve the speed that your output loads for end users in several ways. See Improving the Processing Performance of the Target.
  • Setting the Maximum Concurrent Builds See Setting the Maximum Number of Concurrent Builds.
  • Salesforce® Publishing See About Salesforce® Publishing.
  • Zendesk Publishing See About Zendesk Publishing.
