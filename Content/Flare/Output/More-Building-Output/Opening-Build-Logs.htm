Opening Build Logs

A build log file has an .mclog file extension. The next time you generate the target, the old log file is replaced with a new one.

how to Open a Build Log from the Interface
  1. Open the Builds window pane (View > Builds). See Builds Window Pane.
  2. In the grid select the target whose build log you want to open.
  3. In the local toolbar click Open Build Log.
 Viewing the Log File

The log file contains the following tabs:

  • Messages Tab

  • Warnings Tab If you double-click a row in the log file, the file that is shown in the File column is opened so you can deal with the issue.

    Use caution when ignoring compiler warnings. Only ignore warnings that you are certain you do not need to receive.

  • Errors Tab
