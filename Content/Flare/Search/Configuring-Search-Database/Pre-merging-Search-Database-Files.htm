Pre-merging Search Database Files

If you have multiple projects and plan to merge their outputs when building the "master" project, you can pre-merge the search database files. See Merging Projects.

how to pre-merge search database files
  1. In the Project Organizer, open the target.
  2. Select the Search tab.
  3. Expand the Advanced Search Options section.
  4. Place a check mark in the Pre-merge search database file box.
