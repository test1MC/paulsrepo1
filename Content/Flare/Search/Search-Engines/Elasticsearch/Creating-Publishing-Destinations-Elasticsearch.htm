Creating Publishing Destinations for Elasticsearch

How to Create a Primary Destination

You have the option in the Destination Editor to choose “File System” as the method for transferring files. In this case, you would be storing the copy of your output files to a local folder instead of transferring them remotely. You might choose to do this if you prefer to use a third-party tool to then move those files up to the server hosting your output.

If you choose to publish your files locally in this way, you must have Java Development Kit (JDK) version 8 installed. See https://www.madcapsoftware.com/downloads/java.aspx.
