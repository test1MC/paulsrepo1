Enabling Source Control Status Checks—Team Foundation Server

how to enable source control status checks
  1. Select File > Options. The Options dialog opens.

  2. Select the Source Control tab.

    This tab will not be visible if your project is not yet bound to source control. See Binding a Project to Microsoft Team Foundation Server.

  3. Click the check box Enable background status checks. A check mark in the box indicates that the feature is enabled.
  4. Enter the number of minutes and or seconds between each status update.
 note

If you elect to disable this feature (disabled is the default setting), you can manually check for status updates by refreshing the Pending Changes window pane or Source Control Explorer. See Viewing Files That are Checked Out.
