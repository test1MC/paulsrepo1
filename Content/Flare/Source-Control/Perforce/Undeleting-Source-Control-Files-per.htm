Undeleting Source Control Files—Perforce

Note

You can only undelete files from Microsoft Team Foundation Server and Perforce. When you delete files from other source control providers, they are removed from the source control repository.

 how to undelete a source control file
  1. In one of the window panes (e.g., Content Explorer, Project Organizer, Pending Changes window pane), select the relevant file(s).

    • Ribbon In the File section select Undelete.
