Disconnecting from Source Control—Perforce

How to Disconnect from Perforce

  1. Check out the files you will need to work on while disconnected. See Checking Out Source Control Files.

    • Ribbon In the Project section select Disconnect.

 How to Reconnect to Perforce

    • Ribbon In the Project section select Reconnect.

  2. Submit the files you previously checked out. See Submitting Source Control Files—Perforce.
