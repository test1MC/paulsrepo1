Publishing Branches—Git

You can publish branches in the following ways:

  • Automatic If your current branch is unpublished, pushing, pulling, or synchronizing automatically publishes the branch. See Pushing Files to a Remote Repository—Git, Pulling Files from a Remote Repository—Git, and Synchronizing Source Control Files—Git.
  • Manual Select an unpublished branch in the Source Control Explorer and manually publish it. You can select any branch, even if it is not your current branch.
