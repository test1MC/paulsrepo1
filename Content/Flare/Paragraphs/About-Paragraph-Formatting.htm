About Paragraph Formatting

Following are some of the more common ways that you can format paragraphs. This guide provides steps and details for some of the tasks that are especially important for print-based output (i.e., those listed below with page references after them). For more information about the other tasks, including steps, see the online Help or the Styles Guide.

  • Alignment  See Aligning Paragraphs.
  • Borders   See Setting Borders for Paragraphs.
  • Rounded Borders For most output formats, you can use CSS3 border-radius style properties to create rounded borders on block elements, such as paragraphs and blockquotes.  See Creating Rounded Borders on Paragraphs and Tables .
  • Drop Caps/Initial Caps See Creating Drop Caps Effects and Creating Initial Caps Effects.See Creating Drop Caps Effects.
  • Hyphenation  See Adjusting Hyphenation.
  • Indentation  See Indenting Paragraphs.
  • Line Spacing  See Setting Line Spacing.
  • Short Line Elimination  See Setting Short Line Elimination.
  • Spacing Above/Below  See Setting Paragraph Spacing Above and Below.
  • Transform Effects See Creating Transform Effects.
  • Widow and Orphan Control This feature is used for print-based output. See About Widows and Orphans and Setting Widow and Orphan Control. See Setting Widow and Orphan Control.
