Documents Window View (Doc-To-Help)

The Documents window view displays your source documents. You can open a document by double-clicking it.

Note Right-Click

You can right-click on a document to do the following:

  • Open it.

  • View document properties.

  • Convert Word documents to HTML5.

  • Designate a document as the project glossary.

  • Create a link to the document (choose Copy Link). The link created can be pasted into any document.

 Other Document Tasks

There are other document tasks you can perform outside of the Documents window view. From the Home ribbon (Source group) you can do the following:

  • Assign source templates to your documents. See About Templates and Styles. See the online Help; the Documents, Templates, and Content Guide; or the Making It Look Good Guide.

  • Designate platform, target, and attribute-based conditions. See About Conditions. See the online Help or the Documents, Templates, and Content Guide.

  • View document properties. See Setting Document Properties. See the online Help or the Documents, Templates, and Content Guide.
Note

Normally, you will be working with Word source documents in the Documents window view. If you have a project created with an older version of Doc-To-Help in which you were working with HTML or XML files, you cannot edit those files within Doc-To-Help in this version. You can convert Word source documents to HTML5, but the primary reason for doing this is to then import those files into MadCap Flare.
