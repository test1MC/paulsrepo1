Dynamic Help Window Panel

This window displays relevant Help automatically as you use your mouse to click within or hover over the Doc-To-Help interface. For the complete Help file, click on the Help icon located in the upper-right. 

If you are creating documentation for a software product and would like your interface to include Dynamic Help, see About Context-sensitive Help.

how to Close or open this window panel
  • Close the window by clicking the close icon in the lower-left corner of the window that reads Close Help.
  • Reopen the Dynamic Help window panel by clicking the same icon that now reads Help.
