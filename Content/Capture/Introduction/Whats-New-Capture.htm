Capture Editor Improvements

Improvements have been made to the Capture Editor to make working in the Editor easier. These changes include a centered workspace, a transparent background, and the ability to manually expand the canvas. See Capture Editor.

Crop Enhancements

The crop feature has been significantly enhanced, including the addition of crop features to the context menu. To crop, right-click on the image and select Crop Image, or click on the image object and select Crop Image Object from the context menu. This displays the crop region as well as additional cropping tools, such as Maximize Crop, Crop, and Cancel. See About Cropping.

Drawing Tools

Capture 7 includes three new drawing tools: the pencil, the eraser, and the color fill tool. You can find these tools in the Draw section of the Edit ribbon or in the local toolbar of the Capture Editor. See About Drawing Tools.

Flatten Objects

If you have multiple objects on a canvas, you can now flatten them so they become part of the background. This is helpful when you want an image that can be used as a background for other images. Because flattened objects can no longer be selected or moved, there will be no variations in the background image behind any additional objects added to the image. Additionally, if you are bringing in captures or images from another tool, you can flatten them onto the background before adding callouts or other objects. See Flattening Objects.

Palettes and Profiles Updates

Several enhancements have been made to profile and palette functionality that allow you to easily share these file types with others. See About Palettes and About Profiles.

Selection Tools

Capture 7 includes two new selection tools: the selection rectangle and the magic wand. You can find these tools in the Select section of the Edit ribbon or in the local toolbar of the Capture Editor. Additionally, there are new context menu options available for working with selected areas. See Using the Selection Rectangle and Using the Magic Wand.
