Grouping Objects

You can group multiple objects into a shape group, which then acts as a single object. You can ungroup the objects at any time. Shape groups do not show individual keyframes, effects (excluding shadows), and effect keyframes in the Timeline, Frame Editor, or in the output, but you can apply these things to the group. However, individual objects within the group retain keyframes, effects, and effect keyframes, and those things will once again become editable on each object once they are ungrouped.

Example

Let's say you have an image with three objects—a rectangle, a line segment, and an image object, like this:

You can hold down the CTRL key and select all three objects.

Then you can either click in the local toolbar or right-click and select Group Selected Items, which converts the three objects into a single object group.

 how to group objects
