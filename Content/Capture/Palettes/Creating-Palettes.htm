Creating Palettes

provides you with an initial palette to help get you started. However, you can easily create additional palettes for different purposes.

how to create a palette
    • Ribbon Select View > Palettes.
    • Standard Toolbar Click .

    The Palettes window pane opens.

  2. In the local toolbar of the Palettes window pane, click .
  3. In the Save As dialog, navigate to the folder where you want to save the palette.
  4. In the File name field, enter a name for the palette.
  5. Click Save. The new empty palette opens (by default on the left side of the user interface).
