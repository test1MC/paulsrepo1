Flattening Objects

Flatten lets you merge selected objects with the background image, combining them into a single image. This is useful when bringing in images from another tool, because you can make them a permanent part of your background image. This makes it easier to add and manipulate additional objects.

Example How to Flatten Objects
  1. Open the image to which you have added objects.
  2. In the Capture Editor, select one or more objects.

    • Ribbon In the Order section, select .

    • Layout Toolbar Select .

    • Context Menu Select Flatten.

  4. The object(s) and its associated layer will be merged into the image.

 NoteS

Any objects outside of the true canvas area will be cropped out when the image is flattened. If an object is partially outside of the true canvas area, it will be cropped at the true canvas border.

Once you flatten an object into the image, the object layer is removed. You will not be able to edit the object properties once it has been flattened.
