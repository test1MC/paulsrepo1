Creating Variables

You can create new variables in images, although a more common task is to link to existing variables from a MadCap Flare project.

You can create variables in images using the File Properties dialog or the Profiles Editor. Use the Capture Editor if you want to create variables for a single image only. Use the Profiles Editor if you want to create variables for a profile, which can be used when capturing future images. See About Profiles.

how to create variables using the file properties dialog
  1. Open the image.
  2. Double-click the image. The File Properties dialog opens.
  3. Select the Variables tab.

  4. To enter a new name, definition, or comment for a new variable, click in the appropriate cell. Then press F2 on your keyboard and type the name, definition, or comment. Press Enter after you finish typing. As soon as you begin typing in one of the cells, a new blank row is added in the tab (to be used for the next variable that you create).

    Adding a comment is optional and for your internal use only.

  5. Click OK.
 how to create variables using the profiles editor
  1. Open the profile.
  2. Select the Variables tab.
  3. To enter a new name, definition, or comment for a new variable, click in the appropriate cell. Then press F2 on your keyboard and type the name, definition, or comment. Press Enter after you finish typing. As soon as you begin typing in one of the cells, a new blank row is added in the tab (to be used for the next variable that you create).
  4. Click OK.
 note

In addition to adding a new variable in Capture, you can also link to existing variables in a MadCap Flare project. See Linking to Flare Projects.
