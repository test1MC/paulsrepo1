Opening Profiles

You can open profiles from inside the interface or by dragging the files from a Windows folder.

how to open a profile from within the interface

    • Ribbon In the Tools section, select the Profiles button .

    • Tool Strip Select View > Profiles. 

    • Standard Toolbar Click .

      The Profiles Editor opens.

  2. In the local toolbar, click the down arrow and select the profile that you want to open from the list. The name of the profile displays in the local toolbar of the Profiles Editor.
 how to open a profile by dragging it from windows

  1. Open Windows and navigate to the folder containing your profile (CAPRO file). By default, profiles are stored in your Documents/My Profiles folder.

  2. Launch .

  3. Drag the CAPRO file from Windows to the application window and drop it on the title bar in .

    You can also use this method to open any file type that is supported in .

    You can use this method to open linked profiles as well. See Linking to External Profiles.
