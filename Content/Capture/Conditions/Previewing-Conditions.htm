Previewing Conditions

how to preview conditions
  1. Open an image that you want to preview.
  2. Make sure you have already applied conditions to objects on the image and associated the image with condition tags (including and excluding). See Applying Conditions and Associating Condition Tags with Images.

  3. In the Object section, click the Preview Conditions check box .

    The objects that have condition tags set to be shown still appear, but the objects that have condition tags set to be excluded are hidden.
