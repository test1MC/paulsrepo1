Primary Window Panes

has numerous window panes in the interface that are used for a variety of purposes. The pane that you are likely to use most is the Review Packages Window Pane. Some elements are contained in window panes as opposed to dialog windows because they contain features that you would want to have easy access to as you work in an editor.

Following are the window panes that are probably most commonly used in :

  • Review Packages Window Pane
  • Styles Window Pane
  • SharePoint Explorer
  • External Resources Window Pane
