Formatting Window Pane

The Formatting window pane can be used to work with both styles and local formatting in the open content file.

Style Inspector and Local Style Properties

This window pane consists of two tabs—Style Inspector and Local Style Properties.

  • Style Inspector This feature lets you see the style details for selected content in the open file, and even edit styles for .htm and .mcdoc files, without having to open the full stylesheet. Other file types (e.g., .fltrev, .mcco, mcpgl) are not supported in the Style Inspector. See Using the Style Inspector.

  • Local Style Properties See Local Formatting.

  • Ribbon Select Home > Formatting.
  • Keyboard Shortcut Press CTRL+F12.
