Creating Basic and Advanced Tag Expressions

You might create tag expressions when you are editing a document. See About Condition Tags.

 

note instructions for the basic method

If you want to exclude a condition tag, click the Exclude check box next to it. If you want to make sure a condition tag is included, click the Include check box next to it. The primary reason for having Include check boxes is to account for possible conflicts.
