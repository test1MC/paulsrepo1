PDF Files as Images tasks associated with images

You can accomplish the following with images.

adding and inserting images
  • Images (Insert) These steps show you how to insert an image file that already exists. See Inserting Images.
  • Screen Capture (Insert) Follow these steps if you have MadCap Capture installed on your computer and you want to capture an image from your screen and insert it into a topic at the same time. See Inserting Screen Capture Images.
 editing images
  • Background (Add) See Adding Backgrounds to Images.
  • Borders (Add) See Adding Borders to Images.
  • Image (Move) See Moving Images in Content Files.
  • Image (Resize) See Resizing Images.
  • Margins (Add) See Adding Margins to Images.
  • Padding (Add) See Adding Padding to Images.
  • Position You can position an image so that content wraps around it. See About Positioning Elements.
  • Thumbnails (Preview) See Previewing Thumbnails.
  • Thumbnails (Show While Editing) See Showing Images as Thumbnails While Editing.
 additional image tasks
  • Delete These steps show you how to delete an image that you have previously inserted into a topic. See Deleting Images from Content Files
  • Hyperlink and Image Map (Insert)  See Inserting Image Hyperlinks. You can also create multiple hyperlinks for specific sections of the image, and even place conditions on those sections. This is called an "image map." See Creating Image Maps.
  • Image Frame (Add) You can draw an image frame in a page layout. The image you select is added to a frame of the same size, which you can place on the page as necessary. This lets you place an image automatically on multiple pages in the output. You might use this feature, for example, if you want to place your company logo somewhere on each page. See Adding Images to Page Layout Frames.
  • Select To select an image in the XML Editor, you can left-click on it. Another option is to right-click on it and choose Select from the context menu.
 note

In order for SVG images to display in the XML Editor, you need to install the latest Oracle Java Runtime Environment (JRE) or OpenJDK.
