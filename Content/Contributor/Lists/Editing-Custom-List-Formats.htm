Editing Custom List Formats

After you create custom list formats, you can edit them in a couple of ways:

  • Edit List Format Dialog
  • Styleshseet Editor
how to use the edit list format dialog
  1. Open the document or topic where you have added a list using the custom format.
  3. Right-click the outermost (i.e., first level) structure bar (ol or ul) for the list.

  4. From the context menu, select Type > Custom Format > Edit Custom Format.

  5. Use the Edit List Format make changes to the format.

  6. The dialog closes and any lists associated with that custom format are automatically changed.
