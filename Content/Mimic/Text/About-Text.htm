About Text

You can easily incorporate text into your movies in order to explain the various parts of a movie or to provide instructions for end users.Following are the primary tasks regarding text.

  • Adding Text You can add text to an object (e.g., bubble callout, rectangle) by simply clicking on the object and typing. You can also use the Object Properties window pane. If you want the text to appear by itself, you can edit the properties of the object accordingly so that the object border or background cannot be seen (e.g., specify no border or color for the shape). Otherwise, the object will be shown with the text inside. See Adding Text.

  • Aligning Text  After you add text to an object, you can easily align the text within the object. See Aligning Text.
  • Editing Text After you add text to an object, you can make changes to it as necessary. See Editing Text.
  • Setting Font Properties for Text After you add text to an object, you can adjust the font properties (e.g., font size, color) by using the Text section in the Home ribbon. See Setting Font Properties for Text.
  • Pinning Fonts You can pin your favorite fonts to the top of the font family list so you can easily find them again later. See Pinning Fonts.
  • Font Groups The font family list contains groups for all fonts and pinned fonts. You can expand or collapse each group to limit the list of fonts to those that you use the most and hide from view those that you do not use. See Using Font Groups.
