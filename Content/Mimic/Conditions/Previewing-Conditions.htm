Previewing Conditions

how to preview conditions
  1. Open a frame that you want to preview.
  2. Make sure you have already applied conditions to objects on the frame and associated the movie, collection, or target with condition tags (including and excluding). See Applying Condition Tags and Associating Condition Tags.

  3. In the Object section, click the Preview Conditions check box .

    The objects that have condition tags set to be shown still appear, but the objects that have condition tags set to be excluded are hidden.

 note

Condition tags are also considered when viewing a movie in Preview Mode.
