Example

Let's say you need to create several movies, each of which has an identical opening frame. Perhaps this "Introduction" frame contains an image object showing your company logo, audio, and a button to continue to the next frame.

You can re-create that frame each time you need to produce a new movie, but that can be time consuming. Plus, you want to make sure it is identical to the Introduction frames in all of your other movies.

A better method is to create that Introduction frame just once and then add it to a frame library.

Then, after you create a new movie, you simply open the frame library, right-click on the Introduction frame, and add it to the movie.
