About Master Frames

Example

Let's say that you want to add Back and Next buttons at the bottom of almost all of the frames in your movie. The hard way to accomplish this is to create each button on each frame, or copy and paste them. The easy way is to create one master frame, add the two buttons on it (editing the appearance, button actions, etc.), and then apply that master frame to the relevant frames in the movie.

The easiest way to apply a master frame to all (or most) of the frames in a movie is to do so in the Movie Properties dialog. When you create master frame, you give it a name (or you can use the auto-generated number provided by Mimic). Then, in the Movie Properties dialog, you simply select that master frame.

You can apply the master frame so that it displays on the bottom or top layer of each frame. If it displays on the bottom layer, all other objects on each frame (e.g., buttons, cursors, shapes) will appear in front of the Back and Next buttons from the master frame. If it displays on the top layer, all other objects will appear behind the Back and Next buttons.

Now that the master frame is applied to all frames in the movie, you need to remove it from the first and last frames. To do this, you right-click on each of those frames and select Remove Master Frames.
