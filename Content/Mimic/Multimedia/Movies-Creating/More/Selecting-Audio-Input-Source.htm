Selecting the Audio Input Source

You can record narration while shooting a movie.

how to select the audio input source

  1. Select File > Recording Options.

    The Recording Options dialog opens.

  2. In the Audio Input Source section, select one of the options. This field lists all of the audio input sources found on your computer. Depending on the type of input source, you may need to attach a microphone to your computer. Then, as you record a movie, you can speak into the computer or microphone and this audio is automatically added to the movie frames.

    You can also enable or disable the microphone from the recording task bar.

 Note

As an alternative to narrating while you record a movie, you can add narration to the movie after it has been recorded. See Adding Audio to Movies and Adding Audio to Frames.
