Setting Transitions for Frames

how to set the transitions for a frame
  1. Open the frame.
  3. Expand the Frame Transition section.
  4. Change the settings in the Transition section:

    • Effect Select the effect to be used when the frame transitions at the beginning and end of its play time.

    • Duration Enter the number of seconds that you want the effect to be displayed.

    • Direction Select the direction that you want the effect to move.
note

You can also set the transitions for all frames in a movie. See Setting Transitions for Movies.
