Removing Actions from Objects

If an action is no longer needed, you can remove it from an object.

How to Remove an action from an Object

If you want to remove the visual state from the object but keep it available in case you want to use at another time, you should remove the action from the object.

  1. Open a movie that contains one or more objects.

  2. Double-click an object that has an action applied to it.

  3. Expand the Action section.
  4. Select the visual state you want to remove. The selected action (as well as its associated Event, Shape, and Visual State, if applicable) is highlighted in gray.
  5. Click . The action is removed from the object.

 note

Buttons include a default click action. You should select and delete this click action before creating a new click action.
