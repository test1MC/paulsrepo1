About Actions

An action is a command that you can assign to an object to affect what happens next in the movie. You can also assign an end action to a movie.

Types of Actions

You can apply actions to any object. These actions will commence whenever an event has occurred (e.g., a user clicks a button).

  • Actions After you add an object (such as a button) to a frame, you can assign a regular action to it (e.g., go to next section, pause movie). When the user clicks (or double-clicks) the object in the output, that action will be performed. See Selecting Actions for Objects.
  • End Actions for Cursors If you have cursor shapes in the frames of your movie (either added manually or automatically during the recording), what happens in the movie when a cursor movement reaches the end of its path? Unless you specify otherwise, the movie will simply continue to play in its normal progression. It is likely that this is often what you will want to happen. However, there may be times when you want to provide a specific action to take place when the cursor reaches the end of its path. See Selecting an End Action for a Cursor. For more details, see the online Help.
