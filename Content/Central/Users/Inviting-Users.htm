How to Invite a Single User
  1. On the left side of the interface, click Users.
  2. In the toolbar click .
  3. On the User tab, complete the First Name, Last Name, and Email fields.
  4. From the User Seat Type field, select the kind of user you want to invite:

  5. If you selected the Author seat type, click the Permissions tab, and choose the permissions that you want to give the new user. For details, see Setting User Permissions.

  6. Click Send Invitation. If the invitation is successful, an email is sent to the user. The user must click the link in the email to activate the account. Until users activate the account, they are listed as "Invited" in the system.
How to Invite Multiple Users in Bulk

To use the bulk invitation process, you first need to create or have access to a comma-separated value (CSV) file containing the information for the users you want to add. The CSV file should have three columns, with the following in the top cells of each column:

  • Email
  • FirstName
  • LastName

If you do not yet have such a file, you can download a template from Central, as described in the following steps.

  1. On the left side of the interface, click Users.
  2. In the toolbar click .
  3. From the User Seat Type field, select the kind of user you want to invite:

  4. (Optional) If you do not yet have a CSV file, you can click Download Template.

    This template speeds the process because the necessary column headings are already provided.

    After you complete the cells with each user’s email and name, save the CSV file.

  5. Click Bulk Invite.

  6. Find and select the CSV file. Then click Open.

  7. If you selected the Author seat type, click the Permissions tab at the top, and choose the permissions that you want to give the new users. For details, see Setting User Permissions.

    All of the users in the CSV file will receive the same permissions that you choose. However, you can always edit the permissions for a particular user later.

  8. Click Upload File. The dialog indicates the new users that will be invited. Also, it will show if any of the selected users are already on the license, as well as any invalid emails that are found in the CSV file.
  9. Click Accept & Send Invitations. If the invitations are successful, an email is sent to each user. The user must click the link in the email to activate the account. Until users activate the account, they are listed as "Invited" in the system.
 Notes

If no user seats are remaining, you will be informed. You must purchase more user seats before adding new ones. See Purchasing User Seats and Storage Space.

Alternatively, you can deactivate an existing user to free up a user seat. See Deactivating Users.

For more information, see the online Help.

You can change the user's permissions any time. In addition, you can set permissions globally for all projects or separately for individual projects. See Setting User Permissions.

You cannot set permissions for SMEs.
