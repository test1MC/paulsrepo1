Multiple Page Views

Multiple Page Views

The Tasks page is a single framework with multiple views, which can be accessed at the top of the interface. The different views are:

  • Board (default view)
  • Grid
  • Calendar
  • Backlog
  • Archive

Tasks Activity

Tasks Activity

If you are viewing Central on a larger screen or at a high resolution, there is a tracker on the right side of the Tasks page showing the most recent task activities.

If you are viewing Central on a smaller screen or at a low resolution, the task activity can be accessed from the flyout menu on the left.
