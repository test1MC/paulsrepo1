About Reviews

A streamlined review workflow is available between recent versions of Flare and MadCap Central. This process lets you collaborate with subject matter experts (SMEs) and other authors by viewing and working with the same Flare content files (i.e., topics and snippets) in a lightweight editor.

The Reviews page can be opened on the left side of the interface. From this page, you can view, edit, and manage files that have been sent from MadCap Flare for review.

User Types

User Types

There are two user types in Central—Authors and Subject Matter Experts (SMEs).

When you invite someone to join a Central license, you can choose whether that person will have the Author or Subject Matter Expert user type.

Workflow

Workflow
  1. Original Author—Invite Reviewer to Central License See Inviting Users.
  2. Reviewer—Set Password

  3. Original Author—Select File(s), Choose Reviewer(s), and Send See the Flare online Help.

  4. Reviewer—Open Pending File and Edit/Annotate See Editing and Annotating Review Files.

  5. Reviewer—Submit File See Submitting Review Files.

  6. Author—Close Review File See the Flare online Help.

  7. Author—Accept/Reject Changes See the Flare online Help.

  8. Author—Accept File See the Flare online Help.
