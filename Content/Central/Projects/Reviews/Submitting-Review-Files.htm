Submitting Review Files

Reviewers can return to a file in the Pending tab repeatedly over time to make changes to it. When reviewers finish working on a file and do not need to make any further edits, they submit the file. When all reviewers assigned to the file have submitted it, the original author is informed and can then close out the review process.

How to Submit a Reviewed File
  1. On the left side of the interface, click Reviews.
  2. From the Pending tab on the left, select the file. The file opens in the lightweight editor to the right.

  3. In the upper-right corner, click Submit.
