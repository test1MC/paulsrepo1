Removing or Deleting Termbases

If you are no longer using a termbase, you can remove it.

If you no longer need a termbase or the data in the termbase, you can delete it.

Important how to remove a termbase
  1. In the Termbase section, select Choose Termbase.
  2. Select the termbase that you want to remove.
  3. Click Remove.
 how to Delete a termbase
  1. In the Termbase section, select Choose Termbase.
  2. Select the termbase that you want to delete.
  3. Click Delete.
