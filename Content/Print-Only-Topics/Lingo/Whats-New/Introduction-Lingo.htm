Introduction

Following are the new features in this release of Lingo.

Feature

What's New?

DeepL Machine Translation

  • Choose DeepL for machine translation
  • Translate projects with DeepL

Export All Languages

  • New drop-down option for all languages when exporting
  • New option to append language code to files

Thesaurus

  • New Thesaurus window pane
  • Search for synonyms and apply them to translations

Additional Features

  • Adobe FrameMaker 2019 support
  • Interface themes updated
  • Language drop-down in File List window pane
  • Norwegian Bokmal and Nynorsk language support
  • Source control enhancements (merging and reverting branches)
  • Start Page redesigned
  • Support for new file types (JSON, Flare micro content, SVG)
