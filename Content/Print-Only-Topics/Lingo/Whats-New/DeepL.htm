DeepL Machine Translation

You can now use DeepL as a machine translation (MT) option. You can connect to DeepL through the Options dialog or choose the MT provider through the Resources ribbon. See Choosing a Machine Translation Provider and Setting Machine Translation Options.

Please refer to the DeepL website for a current list of supported languages.

How to Set DeepL for Machine Translation

  1. In the Resources ribbon, click Choose MT. The Machine Translation Providers dialog is displayed.

  2. Click DeepL. The Machine Translation Provider Settings dialog is displayed.

  3. Enter the Authentication Key.
  4. (Optional) Click Test Connection.
