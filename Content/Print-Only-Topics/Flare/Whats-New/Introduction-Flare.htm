Introduction

Following are the new features in this release of Flare.

Feature

What's New?

Micro Content

  • Short, concise information that stands alone and is easily consumable
  • Phrase/response combinations
  • Featured snippets in search results
  • Other possible uses (chatbots, FAQ databases, virtual and augmented reality instructions, field-level Help, etc.)

CSS Variables

  • Set value in one place and reuse throughout stylesheet
  • Helps with speed, ease of use, and consistency
  • Ideal for setting colors

Zendesk Plug-In

  • Publish Clean XHTML or HTML5 skinless output to Zendesk Help Center dashboard
  • Beneficial for knowledge bases or FAQs that support service desk tickets
  • Publish directly to categories and sections

Additional Features

  • Chinese user interface in Flare
  • Elasticsearch enhancements
  • Interface themes—updated
  • Project templates—streamlined and updated
  • Start Page redesigned
  • Stylesheet Editor—ellipsis button for editing values
