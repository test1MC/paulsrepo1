How to Create a Destination File for Zendesk

  1. Open the Project Organizer.
  2. Right-click the Destinations folder and select Add Destination. The Add File dialog opens.

  5. In the File Name field, type a new name for the destination (e.g., FictionSoft Staging Website, FictionSoft Live Website, Network Documentation Folder).
  6. Click Add. The destination is added to the Project Organizer, and opens in the Destination Editor.

  7. In the Type drop-down field, select MadCap Connect for Zendesk.

  8. Click the Activate button.

    1. Enter the license key you were given with your copy of MadCap Connect for Zendesk.
    2. Click Next.
    3. Enter your information:
      • First Name Enter the first name of the user associated with the license key.
      • Last Name Enter the last name of the user associated with the license key.
      • Email Enter the email address associated with the license key.
    4. Click Next.
    5. Select either Internet Activation or Manual Activation.
      • If you chose Manual Activation, follow the instructions on-screen to copy and paste the information in an email to customerservice@madcapsoftware.com.
    6. Click Next.
    7. If you chose Internet Activation, you will receive a confirmation that your MadCap Connect to Zendesk license has been activated. Click Finish.

  9. In the Credentials section, click Login Credentials. This will enable you to connect to your Zendesk dashboard.

  10. After connecting to your Zendesk account, set the following options in the MadCap Connect for Zendesk section.

  11. In the View URL field, provide the location where your Zendesk output should be viewed after it is published.

  12. In the Upload section, select Upload Log File to have it published with the rest of your content in Zendesk.

    • Upload Only Changed Files Select this to only upload updated files.

    • Remove Stale Files Select this to remove any unchanged files. This will archive files in Zendesk.
