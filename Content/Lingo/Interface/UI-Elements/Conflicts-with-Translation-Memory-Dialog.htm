Conflicts with Translation Memory Dialog

This dialog opens if you attempt to import a TMX file and some of the translations in it are different from the translations that already exist in your translation memory database. You can use this dialog to specify whether the translations in your database should be kept, whether to add the new translations from the TMX file, or both. See About Translation Memory Exchange and Importing TMX Files.

  • Go through the steps for importing a TMX file. See Importing TMX Files.

Button/Section

Description

Source

Displays the source string to be translated.

Keep TM

Select this check box to retain the translation that is already in the TM database.

Already in TM

Displays the translated string that is already found in the TM database.

Add New

Select this check box to add the new, proposed translation from the TMX file to the TM database.

Proposed Translation

Displays the translated string that is being proposed as an addition to the TM database.

Keep All Current

Select this check box to retain all of the translated strings that are already in the TM database.

Add All New

Select this check box to add all of the new, proposed translations from the TMX file to the TM database.
