Translation

Lingo offers several features that help you translate a project, including an alignment tool, machine translation, multilingual options, translation memories, and segmentation rules.

  • Translating Files See Translating Files.
  • Machine Translation Machine translation (MT) is translation of text by a computer or translation program, such as Google Cloud Translation. A great benefit of MT is that it allows translators to translate a large amount of content quickly using an automated translation system, and then make any necessary updates to the translation on a case-by-case basis (e.g., using a termbase to replace certain words, fixing translations that are not accurate in context). Then, the translator can add the target segments to an installed TM, improving the speed for future translations as well. See About Machine Translation.
