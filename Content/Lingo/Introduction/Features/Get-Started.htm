Get Started

Lingo’s workflow lets you import a variety of file types, avoid out-of-project file transfers, integrate “must have” features such as translation memory, and produce accurate project statistics, estimates, and reports. You can use Lingo as a CAT (computer-assisted translation) tool, as a translation management solution, or both.
