Using the Find and Replace in Files Feature

how to open the find and replace in files window pane

You do not need to have any files open in order to use this window pane to search for text.

  • Ribbon In the Find and Replace section select Find and Replace in Files.

  • Keyboard Shortcut Press CTRL+SHIFT+F.
 using the find and replace in files window pane using the find results window pane notes

If you are working with a multilingual project, the active target language is used for find and replace. You can change the active target language in the Project Properties dialog. See Project Properties Dialog.

The Find and Replace window pane is disabled when the focus is on the Translation Memory Editor. However, you can use the Quick Find and Quick Replace widgets to find and replace terms in the TM. See Using the Quick Find and Replace Widgets. For more information, see the Lingo online Help.
