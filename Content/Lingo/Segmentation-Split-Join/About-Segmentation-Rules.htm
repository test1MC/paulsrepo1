Applying Segmentation Rules

Applying segmentation rules

After you create segmentation rules, you must make an edit to the file or create an entirely new project. Then you can update the project to apply rules to the segments. See Updating Projects.

After you create segmentation rules, you will need to create a new Align project to apply the segmentation rules to the segments. For this reason, it is recommended that you create segmentation rules in Lingo prior to creating an Align project, and only create new segmentation rules in Align when necessary.

When you create segmentation rules in Align, they are applied to both the source and target segments, if applicable.
