Removing Termbases

If you are no longer using a termbase, you can remove it. This action retains the database, but hides it from the list in your project. You can add the termbase—and its data—back to Lingo later if you need to use it again. If you want to delete the database altogether, see Deleting Termbases.

how to remove a termbase
  1. In the Termbase section, select Choose Termbase.
  2. Select the termbase that you want to remove.
  3. Click Remove.
