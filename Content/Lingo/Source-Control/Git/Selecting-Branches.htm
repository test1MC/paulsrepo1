Switching Branches—Git

There are three ways to switch branches:

  • Source Control Explorer Use this method if you want to pick a branch based on whether it is published or unpublished.
Warning
how to Switch Branches—Submenu

    • Ribbon In the Project section, click the arrow under Branch.

    • Right-Click If you have the File List open, right-click on any file and select Source Control > Project > Switch Branch.
    • Local Toolbar In the local toolbar of the File List, click , then select Project > Switch Branch.

    • Click No to cancel the switch. Commit your changes before attempting the switch again. See Committing Source Control Files—Git.
 how to Switch Branches—Branch Management Dialog

    • Ribbon In the Project section select Branch.

    • Right-Click If you have the File List open, right-click on any file and select Source Control > Project > Branch.
    • Local Toolbar In the local toolbar of the File List, click , then select Project > Branch.

    The Branch Management dialog opens.

  2. Your current branch is shown at the bottom of the dialog.

  1. (Optional) If there are pending modifications on your current branch, a dialog asks if you want to switch branches and discard your modifications.

    • Click No to cancel the switch. Click Close in the Branch Management dialog and commit your changes before attempting the switch again. See Committing Source Control Files—Git.

 how to Switch Branches—Source Control Explorer
  1. In the Pane section select Source Control Explorer. The Source Control Explorer opens.

    • Click No to cancel the switch. Commit your changes before attempting the switch again. See Committing Source Control Files—Git.
