Managing Projects Using the Command Line

What's Next?

After exporting a project, you can inform the author, who can then open the translated project or file and create output for it.

After updating a project, you can open a list of files that require translation (see Viewing a List of Files Requiring Translation). If the Difference column indicates a change was made to a particular file, you can open that file to translate the new text (you may need to drag the divider in the File List window pane to see the Difference column).
