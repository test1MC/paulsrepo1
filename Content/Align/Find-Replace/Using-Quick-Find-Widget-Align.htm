Using the Quick Find Widget

has a Quick Find widget that is integrated into the editor when looking for text in a file that is open.

how to open the quick find widget
  1. Open a file.

    • Ribbon Select Home > Quick Find.

    • Keyboard Shortcut Press CTRL+F.
 how to use the quick find widget

Click to select any of the following filter options:

  • Regular Expressions If you select this option, you can use regular expressions when you look for content.

    about regular expressions

  • Wildcards If you select this option, you can use wildcards when you look for content.

    about wildcards
  • Whole word
  • Match case

If you do not see any search results, be sure that a filter is selected.
