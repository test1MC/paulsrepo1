Choosing a Translation Memory Database

how to choose a tm database
  1. Make sure you have created at least one TM database.

  2. Select Home > Choose TM.

  3. Use the dialog to do any of the following:

    add/remove tm databases to/from grid create new tm databases delete tm database import tmx files Export tmx files

    specify access—read & write versus read only
What's Next?

After you align segments or files, you should upload them to your translation memory database. If you select specific segments and click in the local toolbar, only those translations will be uploaded. If you click in the local toolbar, all translations in the file will be uploaded. See Aligning Segments and Aligning Files.
